# Emotion Detection using Deep Learning

## 📌 Overview
This project detects human emotions in real-time using a webcam.  
It uses a Convolutional Neural Network (CNN) trained on facial images.

## 🎯 Features
- Real-time face detection
- Emotion classification (Happy, Sad, Angry, etc.)
- Webcam integration
- Deep learning model (CNN)

## 🛠️ Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## 📂 Dataset
FER-2013 dataset used for training.

### 2. Train model

### 3. Run detection

## 📸 Output
The system opens webcam and displays detected emotion on face.
